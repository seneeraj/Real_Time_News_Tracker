# 🗞️ Real-Time News Tracker

This is a Python script to fetch and display the latest news using the NewsAPI with the help of streamlit

## Features

- Real-time news on selected topics
- Auto-converts UTC to India Standard Time (IST)
- No external UI framework required

## How to Run

1. Clone the repo
2. Add your NewsAPI key in `news_tracker.py`
3. Run with:

```bash
python news_tracker.py
```

## Dependencies

- requests
- pytz

Install them with:

```bash
pip install -r requirements.txt
```
